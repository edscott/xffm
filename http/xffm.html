Xtended Function File Management

Xtended Function File Management (Xffm) Code Set consists of OpenSource code to enhance filemanagement. It is based on GTK+

Xffm Code Set includes:
  • Interprocess communication: Libtubo
  • Command Search Tool: Fgr
  • Encryption tool: Scramble
  • Graphic differences: Xfdiff
  • Graphic execution: Xffm-run
  • Desktop background setting: Xffm-root
  • File management: Xffm-GUI
    • Desktop GUI
    • Spatial GUI
    • Treeview GUI
    • File Find GUI
  • Xfmime-edit
  • And the Xffm-plugins:
    • Xfsamba
    • Xfbook
    • xffstab
    • xftrash
    • xfapps
    • xfrecent
    • xffrequent
    • xflocate
