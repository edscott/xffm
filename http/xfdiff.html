Rodent-diff: Patch Manager and Difference Viewer

Rodent-diff is graphic interface to the GNU diff and patch commands.

Description
Rodent-diff is graphic interface to the GNU diff and patch commands. With this utility, you can view differences side by side for files or directories. You can also view differences that applying a patch file would imply, without applying the patch. You can also apply patches to the hard disc or create patch files for differences between files or directories. All-in-all, a handy utility for lazy chaps who don't want to type the diff command.
Development
Rodent-diff first appeared around year 2000, as xfdiff, in the xfce3 desktop as an included application. As Xfce transformed into a gnome clone, Rodent applications, including rodent-diff, have continued on the quest for small, fast and powerful applications without heavy baggage.
Although not really necessary, documentation will soon be available.
